# SmartAttendance System

## 📌 Description
A fullstack attendance system using:
- ASP.NET Core Web API
- SQL Server (Stored Procedures)
- Flutter Mobile App

## ⚙️ Features
- Admin & User roles
- Check-in / Check-out
- Location restriction (office only)

## 🛠 Tech Stack
- Backend: ASP.NET Core
- Database: SQL Server
- Frontend: Flutter

## 🚀 How to Run

### Backend
1. Open in Visual Studio
2. Update connection string
3. Run API

### Frontend
1. flutter pub get
2. flutter run
